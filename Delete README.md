# আল-হাদিস ও কুরআন ড্যাশবোর্ড (Bangla Quran Hadith PWA)

একটি আধুনিক, দ্রুত এবং সম্পূর্ণ অফলাইন সমর্থিত ইসলামিক লাইব্রেরি ড্যাশবোর্ড অ্যাপ্লিকেশন। এটি একটি প্রোগ্রেসিভ ওয়েব অ্যাপ (PWA) হিসেবে তৈরি করা হয়েছে, যা আপনি সরাসরি আপনার অ্যান্ড্রয়েড বা আইফোন হোম স্ক্রিনে অ্যাপ আকারে ইনস্টল করে ব্যবহার করতে পারবেন।

## ✨ বৈশিষ্ট্যসমূহ (Features)

* **সম্পূর্ণ অফলাইন সুবিধা:** ইন্টারনেট কানেকশন ছাড়াও অ্যাপটি চমৎকারভাবে কাজ করবে।
* **ডার্ক মোড (Dark Mode):** রাতের বেলা পড়ার সুবিধার্থে রয়েছে দৃষ্টিনন্দন ডার্ক থিম টগল সিস্টেম।
* **৪টি মূল মডিউল:** 
  * 📖 কুরআন ও তাফসীর
  * 🕌 হাদিসসমূহ
  * 📚 ইসলামিক গ্রন্থ
  * 🔖 বুকমার্কস সুবিধা
* **ইনস্টলযোগ্য (PWA):** ব্রাউজারের ৩-ডট মেনু থেকে সরাসরি আপনার মোবাইল ডেস্কটপে যুক্ত করতে পারবেন।

## 🛠️ টেকনোলজি স্ট্যাক (Tech Stack)

* HTML5 & CSS3 (Modern UI Layout)
* JavaScript (Service Worker & Cache API)
* Google Fonts (Hind Siliguri & Amiri)
